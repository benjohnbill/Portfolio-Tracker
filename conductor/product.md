# Product Definition

## Core Purpose
The Portfolio Tracker is a microservices-based application designed to manage quantitative financial portfolios, visualize historical and real-time market data, and provide macro-economic insights.

## Key Features
1. **Real-time & Historical Data:** Fetching prices using Yahoo Finance (US), FinanceDataReader (KR), and FRED (Macro).
2. **Dashboard UI:** Interactive charts and summaries for portfolio tracking and asset allocation.
3. **Quantitative Metrics:** On-the-fly calculation of MDD (Maximum Drawdown), Sharpe Ratio, Beta, and CAGR.
4. **Stress Testing:** Historical crisis simulation and macro-economic yield analysis.

## Phase 2 Acceptance Criteria
- Real-time price updates are implemented on the backend.
- Transaction input form is working on the frontend.
- Quantitative metrics (MDD, Sharpe) are displayed on the Dashboard.
